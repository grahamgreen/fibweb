# fibweb
A simple web app to return the Fibonacci sequence  
###Requirements:
##### from apt-get:
git
pip  
libffi-dev 
libssl-dev
python-dev
#####from pip:
virtualenv  
virtualenvwrapper  

###Build:  

```
mkvirtualenv fib1
git clone https://github.com/grahamgreen/fibweb.git 
cd fibweb
pip install -r requirements.txt
python fib.py
```
